What is Tinkercad?

A free, browser-based design platform by Autodesk

Tinkercad is a free online tool made by Autodesk — the same company behind professional software used by engineers and architects around the world. The cool part? Tinkercad is designed specifically for beginners, so you don't need any experience to start creating.

It runs completely in your browser — no downloads, no installations needed. You log in, and you're ready to create. Everything you make is saved automatically in the cloud, so you can access your projects from any computer.

Tinkercad isn't just one tool — it's actually three different creative environments in one place. Let's look at what each one does.
🧊
3D Design

Design in 3D

Create three-dimensional models by combining and modifying geometric shapes. This is the main tool we'll use in class.

  • Design objects for 3D printing
  • Build prototypes and models
  • Create architectural structures
  • Design LEGO-style builds
  • Export designs to Minecraft
⚡
Circuits

Build Circuits

Design and simulate real electronic circuits — including Arduino projects — directly in your browser, no physical hardware required.

  • Build circuits with LEDs, sensors, resistors
  • Simulate Arduino projects
  • Test designs safely before building them
  • Learn electronics without risk
🧩
Codeblocks

Code Your Designs

Use visual drag-and-drop code blocks (similar to Scratch) to write programs that automatically generate 3D shapes and patterns.

  • Create patterns and shapes with code
  • Generate complex 3D geometry automatically
  • Learn programming logic visually
  • Bridge into text-based coding
🌍 Tinkercad is used by millions of students, teachers, and makers around the world — and it is completely free for everyone, forever.

💡 What is SVG?

SVG stands for Scalable Vector Graphic. Unlike regular images (JPG, PNG) that are made of pixels, SVG files are made of mathematical shapes and lines — which means they can be scaled to any size without losing quality. In Tinkercad, you can import an SVG file and it will automatically turn it into a 3D shape you can use in your designs.

📌 Follow the video step by step — do exactly what it shows. You will use Gemini AI to generate a cat face image, convert it to SVG, and then import it into Tinkercad to build your keychain.
